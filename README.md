# 🌩️ WeatherScope Pro – Modern Weather Dashboard

## 📌 Project Overview

WeatherScope Pro is a modern and fully responsive weather dashboard that provides real-time weather information through an elegant and interactive user interface.

The application integrates with the Open-Meteo API to deliver accurate weather forecasts, environmental metrics, and hourly/daily weather predictions while maintaining a visually appealing storm-themed design.

---

## ✨ Features

- 🌍 Search weather for any city worldwide
- 🌡️ Real-time temperature updates
- 💨 Wind Speed Monitoring
- 💧 Humidity Tracking
- 📈 Atmospheric Pressure Data
- ☀️ UV Index Information
- 🌅 Sunrise & Sunset Timings
- ⏰ Hourly Weather Forecast
- 📅 7-Day Weather Forecast
- ⚡ Dynamic Lightning Animations
- ☁️ Animated Cloud Backgrounds
- 📱 Fully Responsive Design

## 🌐 API Integration

This project uses the Open-Meteo API for weather forecasting.

### Services Used

#### Geocoding API

Converts city names into geographical coordinates.

#### Weather Forecast API

Provides:

- Current Weather
- Hourly Forecast
- Daily Forecast
- Sunrise/Sunset Data
- UV Index
- Wind Information

No API key is required.
